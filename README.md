## 快速开始

### 获取 Account 和 Cookie

在[校园卡网站](https://card.xjtu.edu.cn)（目前学校运营商网络无法访问，可以使用流量或者校园网访问）获取account和hallticket。

### 运行

从 Release 下载对应系统的二进制文件，即可从终端运行。

```bash
# macOS, Linux
./xjtu-mealflow
```

```powershell
# Windows
.\xjtu-mealflow.exe
```
